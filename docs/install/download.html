Загрузка дистрибутива кластера¶

Для загрузки дистрибутива кластера Arenadata Hadoop (далее – ADH) необходимо заполнить Анкету пользователя на веб-странице (Рис.1.): http://arenadata.io/adru-download.html#arenadata-ru-download-easydown

Рис.1. Анкета пользователя

Для заполнения Анкеты следует указать следующие данные:

  • Имя – имя пользователя (обязательное для заполнения поле);
  • Фамилия – фамилия пользователя (обязательное для заполнения поле);
  • E-mail – адрес электронной почты пользователя (обязательное для заполнения поле);
  • Компания – наименование компании пользователя (обязательное для заполнения поле);
  • С какой целью будет использоваться дистрибутив – описание целей использования кластера (обязательное для заполнения поле);
  • Комментарии – комментарии пользователя (необязательное для заполнения поле).

По завершению заполнения Анкеты пользователя необходимо нажать кнопку Отправить. При корректном вводе данных открывается ссылка для загрузки следующих компонентов дистрибутива (Рис.2.):

  • Arenadata Ambari – средство развертывания, настройки и мониторинга сервисов;
  • Arenadata Hadoop 1.4 – RPM-пакеты для сервисов Hadoop, таких как HDFS, YARN, Hbase, Hive, Zookeeper;
  • Arenadata Hadoop Utils 1.4 – дополнительные сервисы и библиотеки, используемые для мониторинга и оповещения серверов кластера.

Рис.2. Страница загрузки дистрибутива

Для старта загрузки необходимо кликнуть по требуемому компоненту, при этом начинается скачивание дистрибутива стандартными средствами Windows.

После загрузки дистрибутивов необходимо установить их, действуя в соответствии с последующими разделами настоящего документа.

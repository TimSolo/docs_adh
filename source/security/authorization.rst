Установка Ranger с помощью Ambari
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

**Apache Ranger** можно установить вручную, используя платформу **Arenadata Hadoop**, или с при помощи пользовательского интерфейса **Ambari**. В отличие от ручного процесса установки, требующего выполнения ряда шагов, установка **Ranger** с использованием интерфейса **Ambari** проще и легче. Опция службы **Ranger** доступна через мастер **Add Service** после инсталляции кластера **ADH** с помощью установщика.

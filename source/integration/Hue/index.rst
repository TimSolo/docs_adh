Использование Hue совместно с Arenadata Hadoop
===============================================

Инструкция может быть полезна администраторам, программистам, разработчикам и сотрудникам подразделений информационных технологий, осуществляющих пользовательскую работу с сервисами Hadoop, таких как HDFS, Hive, YARN, HBase, Spark, Zookeeper.

Кластер Arenadata Hadoop (ADH) устанавливается при помощи `Arenadata Cluster Manager <https://arenadata.tech/products/adcm/>`_ (ADCM) -- отдельного продукта Arenadata, предназначенного для простого, удобного и быстрого развертывания и эксплуатации программного обеспечения.

С документацией ADCM возможно ознакомиться по следующей ссылке: https://docs.arenadata.io/adcm/

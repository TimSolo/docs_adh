Руководство по работе с Apache Zeppelin
=======================================


В документе приведено общее описание Apache Zeppelin, руководство по установке, конфигурации, использовании интерфейса и интерпретаторах. 

Документ может быть полезен администраторам, программистам, разработчикам и сотрудникам подразделений информационных 
технологий, осуществляющих внедрение и сопровождение кластеров Arenadata Hadoop и Arenadata DB.

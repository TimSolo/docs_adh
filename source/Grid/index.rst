Руководство по работе с Arenadata Grid
======================================

В документе приведено общее описание Arenadata Grid: основные концепции, Durable Memory и хранилище "ключ-значение", загрузка данных и стриминг. 

Документ может быть полезен администраторам, программистам, разработчикам и сотрудникам подразделений информационных 
технологий, осуществляющих внедрение и сопровождение кластеров Arenadata.

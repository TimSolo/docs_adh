Поддерживаемые шлюзом сервисы Hadoop
=====================================

Шлюз **Knox Gateway** поддерживает следующие сервисы **Hadoop** в кластерах как с **Kerberos**, так и без:

+ Поддерживаемые API-интерфейсы компонентов (прокси-сервер):

  + YARN
  + WebHDFS
  + WebHCat/Templeton
  + Oozie
  + HBase/Stargate
  + Hive (через WebHCat)
  + Hive (через JDBC)
  + Ambari
  + Atlas
  + Ranger
  + Zeppelin

+ Поддерживаемые пользовательские интерфейсы компонентов (прокси-сервер):

  + Ambari UI
  + Atlas
  + Ranger Admin Console
  + Zeppelin

.. important:: API и визуальные интерфейсы Knox, которые не перечислены в данной главе, рассматриваются как функционал, добавляемый Community проекта

.. important:: Функционал Community проекта разрабатывается и тестируется, но официально не поддерживается компанией Arenadata. Эти функции исключаются по ряду причин, включая недостаточную надежность или неполное покрытие тестового сценария. Подобные функции не рекомендуется использовать в рабочей среде

Руководство пользователя по работе с HDFS
==========================================

Руководство является отправной точкой для пользователей, работающих с распределенной файловой системой **Hadoop** (**HDFS**, Hadoop Distributed File System) либо в составе кластера **Hadoop**, либо в качестве автономной распределенной файловой системы общего назначения. Хотя **HDFS** предназначена для "just work" во многих средах, практические знания о файловой системе могут значительно улучшить конфигурацию и диагностику на определенном кластере.

Руководство может быть полезно администраторам, программистам, разработчикам и сотрудникам подразделений информационных технологий, осуществляющих внедрение и эксплуатацию кластера.

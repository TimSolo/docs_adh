Глоссарий
=========

.. glossary::
   :sorted: 
   
       
   Arenadata Unified Data Platform
      Универсальная платформа данных – это интегрированный набор компонентов 
      корпоративного уровня на базе решений с открытым исходным кодом.   

   Исходный код
      Текст компьютерной программы на каком-либо языке программирования или языке разметки, который может быть прочтён человеком. В обобщённом смысле — любые входные данные для транслятора.
